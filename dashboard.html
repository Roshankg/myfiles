Instructions:

  • Select Create from left menu to create a new service
  • Select Calibrate from left menu to Calibrate
  • Select Containerize from left menu to Containerize

Prerequisites:

  1. Download JDK 1.8 version. Please ensure the System Environment Variables are properly set pointing to JDK 1.8 if you already have an older version of JDK installed on system.
  2. Download Eclipse. This tool is recommended IDE which can be used for development. You are free to use other IDEs like STS, IntelliJ IDEA, etc..
  3. Download Maven. This tool is required to build the Java Projects.
  4. Download Git. This tool is required to download the available Microservices from GitHub Repository. This can be used for check in and check out the code using Github repository.
  5. Download Cloud Foundry CLI (Command Line Interface). This tool is required to push the application to Pivtoal Cloud Foundry.
